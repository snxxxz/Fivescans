About Fivescans

A sophisticated tool designed to help server administrators, developers, and security researchers analyze FiveM server dumps with precision and insight.

How It Works

Server Events

Identifies all TriggerServerEvent calls and parameters.

Client Events

Extracts all TriggerEvent calls for local client interactions.

Coordinates

Locates coordinates in vector3 format or table format with contextual information.

Features

  • Fast, client-side analysis of dump files
  • Intelligent parameter type detection
  • Coordinate discovery and association
  • Context-aware code analysis
  • Exportable results for later use
  • Responsive design with dark mode

Ethical Usage

Fivescans is intended for educational purposes, security research, and server administration on servers you own or have explicit permission to analyze. Unauthorized scanning or analysis of servers is strictly prohibited. Always respect privacy and legal boundaries.
